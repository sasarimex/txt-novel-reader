# 摸鱼神器 界面原型说明

版本：P0 v0.8  
日期：2026-05-28  
平台：Windows 桌面  

## 1. 设计原则

- 第一屏就是可用书架，不做营销页。
- 界面安静、低干扰，服务长时间阅读。
- 阅读器内所有设置入口保持克制，避免遮挡正文。
- 翻页模式必须互斥，用户在阅读时不会被两套翻页逻辑同时影响。
- 所有弹窗都要明确当前动作，尤其是覆盖和删除。

## 2. 页面结构

P0 包含两个主页面：

- 书架页
- 阅读器页

阅读器页包含两个主要弹窗：

- 目录弹窗
- 设置弹窗

系统操作包含两个确认弹窗：

- 重复导入弹窗
- 删除确认弹窗

## 3. 书架页

### 3.1 书架页布局

```text
┌────────────────────────────────────────────────────────────┐
│ 摸鱼神器                                      导入 TXT     │
├────────────────────────────────────────────────────────────┤
│ 排序: 最近阅读 v                                           │
│                                                            │
│ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐        │
│ │  诡秘之主    │ │  凡人修仙传   │ │  剑来        │        │
│ │  进度 32%    │ │  进度 4%     │ │  进度 88%    │        │
│ │          ⋯   │ │          ⋯   │ │          ⋯   │        │
│ └──────────────┘ └──────────────┘ └──────────────┘        │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 3.2 顶部栏

左侧：

- 产品名：`摸鱼神器`

右侧：

- `导入 TXT` 按钮

### 3.3 排序控件

位置：

- 顶部栏下方左侧。

选项：

- 最近阅读，默认。
- 阅读进度。

行为：

- 最近阅读：按最近阅读时间倒序。
- 阅读进度：按阅读进度从低到高排序。

### 3.4 书籍卡片

卡片内容：

- 书名。
- 阅读进度，例如 `进度 32%`。
- 更多按钮，用于删除。

交互：

- 点击卡片主体：打开阅读器。
- 点击更多按钮：显示删除入口。
- 点击删除：弹出删除确认弹窗。

### 3.5 空书架状态

当没有导入任何书时：

```text
┌────────────────────────────────────────────────────────────┐
│ 摸鱼神器                                      导入 TXT     │
├────────────────────────────────────────────────────────────┤
│                                                            │
│                    还没有导入小说                          │
│                    导入 TXT 后会显示在这里                  │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

```text
├────────────────────────────────────────────────────────────┤

│   正文内容正文内容正文内容正文内容正文内容正文内容正文内容                  │
│                                                            │
│                                                            │
│                                                32 / 418    │
└────────────────────────────────────────────────────────────┘
```

### 4.2 顶部栏

左侧：

- 返回书架按钮：`← 书架`

中间：

- 当前书名。

右侧：

- `目录` 按钮。
- 齿轮设置按钮。

要求：

- 顶部栏点击不触发翻页。
- 标题过长时省略，不挤压右侧按钮。

### 4.3 正文区域

正文区域占据主要空间。

默认阅读样式：

- 字号：20px。
- 行高：建议 1.8。
- 正文最大宽度：建议 760px 到 900px。
- 正文居中显示。

说明：

- P0 虽然是 Windows only，但需要适配 13 寸笔记本窗口。
- 正文不应贴边。
- 页码固定在右下角，不能覆盖正文关键内容。

### 4.4 页码

位置：

- 阅读器右下角。

格式：

```text
32 / 418
```

行为：

- 滚轮模式下根据滚动位置换算页码。
- 点击翻页模式下显示真实分页页码。
- 字号或窗口变化时重新计算。

## 6. 设置弹窗

### 6.1 设置布局

```text
┌──────────────────────────────────────────┐
│ 设置                                  ×  │
├──────────────────────────────────────────┤
│ 翻页模式                                 │
│ [ 滚轮模式 ] [ 点击翻页 ]                 │
│                                          │
│ 字号                                     │
│ 16  [-]  20px  [+]  32                   │
│                                          │
│ 主题                                     │
│ [ 白色 ] [ 黑色 ] [ 护眼 ]               │
└──────────────────────────────────────────┘
```

### 6.2 翻页模式

控件：

- 分段控制。

选项：

- 滚轮模式。
- 点击翻页。

互斥行为：

- 选择滚轮模式后，左右半屏点击不翻页。
- 选择点击翻页后，滚轮不滚动、不翻页。

### 6.3 字号设置

控件：

- 减号按钮。
- 当前字号文本。
- 加号按钮。

范围：

- 16px 到 32px。

默认：

- 20px。

行为：

- 调整后立即应用。
- 页码重新计算。
- 阅读位置保持在原位置附近。

### 6.4 主题设置

控件：

- 三个主题按钮。

主题：

- 白色。
- 黑色。
- 护眼。

护眼主题：

- 背景 `#DCE8D2`。
- 文字 `#243024`。

## 7. 重复导入弹窗

触发：

- 用户导入的 txt 书名与书架中已有书名重复。

```text
┌──────────────────────────────────────────┐
│ 书籍已存在                               │
├──────────────────────────────────────────┤
│ 书架中已存在《诡秘之主》，是否跳过？       │
│                                          │
│                    [ 否，覆盖 ] [ 是，跳过 ] │
└──────────────────────────────────────────┘
```

按钮行为：

- 是，跳过：关闭弹窗，不导入。
- 否，覆盖：替换旧书，重置进度，重新解析。

说明：

- 为避免误操作，按钮文案同时保留“是/否”和具体动作。

## 8. 删除确认弹窗

触发：

- 用户在书籍卡片更多菜单中选择删除。

```text
┌──────────────────────────────────────────┐
│ 删除书籍                                 │
├──────────────────────────────────────────┤
│ 确定要删除《诡秘之主》吗？此操作不会删除原始文件。 │
│                                          │
│                         [ 取消 ] [ 删除 ] │
└──────────────────────────────────────────┘
```

按钮行为：

- 取消：关闭弹窗。
- 删除：删除软件内书籍数据和副本。

## 9. 视觉规格

### 9.1 色彩

白色主题：

- 背景：`#F7F7F3`
- 正文：`#202124`
- 辅助文字：`#6B6F76`
- 分割线：`#DDDCD6`

黑色主题：

- 背景：`#101214`
- 正文：`#D8D8D8`
- 辅助文字：`#9A9A9A`
- 分割线：`#2A2D30`

护眼主题：

- 背景：`#DCE8D2`
- 正文：`#243024`
- 辅助文字：`#596A54`
- 分割线：`#BCCAB2`

### 9.2 字体

默认字体：

- 使用系统中文字体。
- Windows 建议优先 `Microsoft YaHei UI` 或 `Microsoft YaHei`。

正文：

- 默认 20px。
- 行高 1.8。

界面文字：

- 12px 到 16px。

### 9.3 间距

书架：

- 页面内边距 24px。
- 卡片间距 16px。

阅读器：

- 顶部栏高度 48px。
- 正文左右边距根据窗口自适应。
- 正文最大宽度建议 900px。

弹窗：

- 宽度建议 520px 到 640px。
- 最大高度建议为窗口高度的 80%。
- 内容超出时弹窗内部滚动。

## 10. 关键交互规则

- 导入成功后留在书架。
- 打开书籍时更新最近阅读时间。
- 退出阅读器时保存当前位置。
- 阅读位置变化后节流保存，避免频繁写磁盘。
- 目录和设置弹窗打开时，底层正文不响应点击翻页。
- 点击翻页模式下，滚轮事件必须被阻止。
- 滚轮模式下，左右半屏点击区域不启用。
- 删除书籍不影响用户原始 txt 文件。
