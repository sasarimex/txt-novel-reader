# GitHub 上传检查清单

## 通用

- [ ] 没有 `node_modules/`
- [ ] 没有 `dist/`
- [ ] 没有 `release/`
- [ ] 没有 `.env` 或 `.env.local`
- [ ] 没有个人 TXT 小说文件
- [ ] 没有 token、密码、API key 或私钥
- [ ] README 已更新
- [ ] `.gitignore` 已更新
- [ ] `npm install` 可恢复依赖
- [ ] `npm run build` 可通过

## 客户端额外检查

- [ ] Windows 安装包只上传到 GitHub Releases
- [ ] 没有把 `.exe` 直接提交到源码仓库
- [ ] `release-assets/` 只作为本地整理目录
- [ ] 云端同步地址示例使用占位域名
