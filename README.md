# 发布说明

## v1.0

首个可用版本，提供本地 TXT 阅读器的核心闭环：导入、解析、阅读、记录进度和管理书架。

### 新增功能

- 支持从本地选择并导入 `.txt` 文件。
- 导入时自动复制文件到应用管理目录，避免直接修改原始文件。
- 支持 UTF-8、UTF-8 BOM、GBK/GB18030 文本解码。
- 支持常见中文卷章标题识别，并生成目录。
- 书架支持展示阅读进度、字数和最近阅读时间。
- 书架支持按最近阅读和阅读进度排序。
- 阅读器支持滚动阅读和点击翻页。
- 支持字号调节、浅色主题、深色主题和护眼主题。
- 支持自动保存阅读位置、页码、总页数和进度。
- 支持目录弹窗跳转到指定章节。
- 支持书签列表和书签位置跳转。
- 支持删除应用内书籍记录和管理副本。

### 工程改进

- 使用 Electron 主进程处理本地文件读取、复制、删除和书库持久化。
- 使用 preload 暴露受控的阅读器接口，渲染进程不直接访问 Node 能力。
- 使用 React 维护书架、阅读器、设置弹窗和目录弹窗状态。
- 对长文本阅读区域做了分段渲染，降低超长小说的渲染压力。
- 对窗口尺寸变化和字号变化做了防抖处理，减少分页和定位抖动。
- 增加目录解析、编码处理和书库读写相关脚本，便于本地回归验证。


### 已知限制

- 当前主要面向 Windows 使用场景。
- 仅支持 TXT 文本阅读，暂不支持 EPUB、PDF 等格式。
- 自动目录识别依赖常见中文章节格式，特殊排版文本可能需要后续优化。
- 文档目录保留了历史草稿，部分早期文件可能存在编码显示异常。
